Module Bitpath_cover_map

module Bitpath_cover_map: sig .. end
The Bitpath_cover_map implements a mappings from sets of infinite bit-strings, representable by a finite set of common prefixes. In other words, the domain has the same structure as the sets modelled by Bitpath_cover. Both polymorphic and a functor variants are provided to abstract over the codomain, though the former only contains the part of the API which do not depend on equality.
